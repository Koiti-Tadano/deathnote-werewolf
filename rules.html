ルール

章ごとに開閉できます。プレイ中の参照にも。

初心者版 ルールブック

1. ゲーム概要

プレイヤーは仮想の村に住む住民となり、「人狼（デスノート所持者）」と「市民」に分かれて正体を隠しながら生き残りを目指す。人狼は他人の仮想フルネームを知ることで、その夜に相手を殺せる。市民は協力して人狼を見つけ出し、10日以内に排除すれば勝利となる。

2. プレイヤー人数
  • 推奨：6〜8人（今回は7人例）
  • 人狼：1人 / 市民：複数 / 狂人：1人
3. 勝利条件

市民陣営

  1. 人狼を吊る
  2. 10日経過時点で人狼が能力を失っている
  3. 人狼が「死神の目」を使用後、5日経過時に人狼が勝利条件を満たしていない

人狼陣営

  1. 市民の人数を人狼側が上回る・同数にする
  2. 能力を保持している状態で勝利条件を満たしている

狂人：人狼が勝利条件を満たす（狂人死亡後でも可）

4. 役職と能力

市民

  • 特殊能力なし
  • 他市民と協力して人狼を排除する
  • 協力には自分のフルネームを明かす必要がある（特殊ルール①）

探偵

  • 期間を「1〜3日目」「4〜6日目」「7日目以降」に区切り、それぞれ1回ずつ能力を使用可能
  • 夜に、その日を含む期間中の、誰か1人の発言を1つ指定し、真偽をGMに確認できる（客観的事実のみ）
  • 判定は「事実だった」or「事実ではなかった」のみ（推測や意見は対象外）

人狼

  • 名前を知った相手を夜にキル可能
  • 一度だけ「死神の目」可能（発動から5日以内に勝てなければ敗北）（特殊ルール②）

狂人

  • 寿命制限付き（ゲーム開始から5日で脱落）で人狼陣営を勝たせる第三陣営
  • ゲーム開始時点で全員のフルネームを知っている
  • 人狼に教えられる名前は2人まで
  • 人狼陣営にはカウントされない（人狼勝利条件は変わらない）
  • 狂人は吊りで殺された場合のみ全体に役職が明かされる
  • 人狼とは互いに正体がわからない
5. 特殊ルール

① 情報カード解除

  • 全プレイヤーは自身の「名刺」と、人狼につながる暗号の「情報カード」を持つ
  • 狂人の情報カードのみ偽物（見分け不可）
  • 交渉2パターン：
    1. 口頭でフルネームを伝える → 相手の情報カードの内容を口頭で聞ける
    2. 名刺を見せる → 相手の情報カードそのものを見せてもらえる

② 死神の目

  • 人狼がゲーム中に一度だけ発動可能。任意の1人のフルネームを知る

③ 捜査グループ

  • 名刺を交換したプレイヤー同士3人以上で作成可（最大4人）
  • グループ内で自由会話・情報カード開示ができる
  • 昼フェーズのいつでも作成可
6. フェーズ進行
  1. 朝：前夜の犠牲者発表
  2. 進行昼（個別会話）：1対1で会話。名前開示は任意
  3. 夕方（全体会議）：情報共有／吊り投票（任意）
  4. 夜：①人狼の能力使用 → ②探偵の能力使用
7. 推奨構成（7人例）
  • 人狼：1人 / 市民：4人 / 探偵：1人（市民陣営） / 狂人：1人
＋α.追加ルール 心理戦のスパイスとしてご自由にトッピングください。

①譲渡能力

  • 人狼は他市民に能力を移すことができる
  • 譲渡できる能力はキル能力のみであり、その他の「死神の目」、「譲渡能力」、「転化能力」は元人狼、新人狼共に使えなくなる（能力が元人狼に戻ると、再び使えるようになる）
  • 譲渡後は能力を持つ者が「人狼」となり、勝利条件も人狼そのものになる
  • 最終的に能力を持っていないと敗北になる
  • 能力を失った人狼は譲渡先のプレイヤーが持っていた情報カードを手に入れ、市民を装うことができる
  • 譲渡先が死亡すると、能力は元の持ち主に戻り、持っていた情報カードは破棄される
  • 元人狼は譲渡しても「死神の目」の寿命カウントは進み、元人狼が死んだ状態で新人狼がキルされると市民の勝利になる

②転化能力

  • 市民陣営の死者が1人以上、残り人数が4人以上の時のみ、死んだ市民1人を復活させる代わりに、1人の市民を「新狼」として人狼に変えることができる
  • 新役職:新狼 　
    1. 転化された時点で人狼陣営の勝利条件に切り替わる
    2. 寿命は転化後3日目夜まで。寿命を迎えると死亡し、翌朝「新狼だった」と公開される
    3. キル可能なのは転化後2日目と3日目のみ。3日目は複数人を同時にキルできる。
      4. ※新狼が誕生した夜の翌朝からを1日目とする
  • 新狼のキルについて 　
    1. 吊りの場合
      2. 新狼を吊った翌朝、正体が明かされる
    2. 寿命の場合
      3. 転化後3日が経過し新狼が脱落すると、翌朝正体が明かされる
    3. 人狼のキルによる場合
      4. 人狼は新狼転化後1日目、2日目の夜に新狼を無条件でキル可能。その場合は翌朝に新狼の正体は明かされない。（3日目夜はキル不可）

    • ③ランダムイベント:デスノートのかけら

  • 発動日
    • 6日目か7日目の朝（GMがランダムで発動）
  • 内容
    • GMがランダムで市民1人に「デスノートのかけら」を渡す（狂人・人狼は対象外）
  • 効果
    • 紙に人狼だと思う相手の正確なフルネームを書き、殺すことができる
  • 使用可能期間
    • 受け取ってから2日間
  • 特殊ルール
    1. 名前が書かれた瞬間、人狼にのみ匿名で「自分が書かれた」ことが通知される。
    2. 書かれてから翌日同時刻に対象が死亡する。
    3. 使用タイミングは朝・昼・夕方・夜のいつでも可（死亡タイミングも同時刻に発生）
    4. 書かれた対象が人狼でない場合は、その人物のみ死亡しゲーム続行

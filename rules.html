初心者版 ルールブック

1. ゲーム概要

プレイヤーは仮想の村に住む住民となり、「人狼（デスノート所持者）」と「市民」に分かれて正体を隠しながら生き残りを目指す。人狼は他人の仮想フルネームを知ることで、その夜に相手を殺せる。市民は協力して人狼を見つけ出し、10日以内に排除すれば勝利となる。

2. プレイヤー人数
  • 推奨：6〜8人（今回は7人例）
  • 人狼：1人 / 市民：複数 / 狂人：1人
3. 勝利条件

市民陣営

  1. 人狼を吊る
  2. 10日経過時点で人狼が能力を失っている
  3. 人狼が「死神の目」を使用後、5日経過時に人狼が勝利条件を満たしていない

人狼陣営

  1. 市民の人数を人狼側が上回る・同数にする
  2. 能力を保持している状態で勝利条件を満たしている

狂人：人狼が勝利条件を満たす（狂人死亡後でも可）

4. 役職と能力

市民

  • 特殊能力なし
  • 他市民と協力して人狼を排除する
  • 協力には自分のフルネームを明かす必要がある（特殊ルール①）

探偵

  • 期間を「1〜3日目」「4〜6日目」「7日目以降」に区切り、それぞれ1回ずつ能力を使用可能
  • 夜に、その日を含む期間中の、誰か1人の発言を1つ指定し、真偽をGMに確認できる（客観的事実のみ）
  • 判定は「事実だった」or「事実ではなかった」のみ（推測や意見は対象外）

人狼

  • 名前を知った相手を夜にキル可能
  • 一度だけ「死神の目」可能（発動から5日以内に勝てなければ敗北）（特殊ルール②）

狂人

  • 寿命制限付き（ゲーム開始から5日で脱落）で人狼陣営を勝たせる第三陣営
  • ゲーム開始時点で全員のフルネームを知っている
  • 人狼に教えられる名前は2人まで
  • 人狼陣営にはカウントされない（人狼勝利条件は変わらない）
  • 狂人は吊りで殺された場合のみ全体に役職が明かされる
  • 人狼とは互いに正体がわからない
5. 特殊ルール

① 情報カード解除

  • 全プレイヤーは自身の「名刺」と、人狼につながる暗号の「情報カード」を持つ
  • 狂人の情報カードのみ偽物（見分け不可）
  • 交渉2パターン：
    1. 口頭でフルネームを伝える → 相手の情報カードの内容を口頭で聞ける
    2. 名刺を見せる → 相手の情報カードそのものを見せてもらえる

② 死神の目

  • 人狼がゲーム中に一度だけ発動可能。任意の1人のフルネームを知る

③ 捜査グループ

  • 名刺を交換したプレイヤー同士3人以上で作成可（最大4人）
  • グループ内で自由会話・情報カード開示ができる
  • 昼フェーズのいつでも作成可
6. フェーズ進行
  1. 朝：前夜の犠牲者発表
  2. 進行昼（個別会話）：1対1で会話。名前開示は任意
  3. 夕方（全体会議）：情報共有／吊り投票（任意）
  4. 夜：①人狼の能力使用 → ②探偵の能力使用
7. 推奨構成（7人例）
  • 人狼：1人 / 市民：4人 / 探偵：1人（市民陣営） / 狂人：1人
